# Public Assets

Place your 3D models (`.glb` files) or static images in this directory.

For example, if you place a file named `coffee-cup.glb` in this folder, you can access it in the application at the path:
`/coffee-cup.glb`

You can also toggle and configure this in the Admin settings:
- **Go to Admin Panel** -> **Settings**
- Turn on **"Use Custom 3D Model"**
- Enter the model path (e.g. `/coffee-cup.glb`)
